# FXLab Frontend

## Prerequisites

This frontend requires Node.js and npm to be installed.

### Installation

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## Testing

The test `tests/unit/test_m0_frontend_structure.py::test_ac8_npm_build_succeeds` requires npm to be available in your PATH.

To install Node.js and npm:
- macOS: `brew install node`
- Ubuntu/Debian: `sudo apt-get install nodejs npm`
- Windows: Download from https://nodejs.org/
